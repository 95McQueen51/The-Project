# Веб-приложение для визуализации прогресса студентов

## Описание проекта
Данный проект представляет собой веб-приложение для анализа и визуализации учебного прогресса студентов.  
Приложение позволяет получать данные об успеваемости, рассчитывать основные метрики и отображать их в виде интерактивных графиков.

Проект состоит из backend и frontend части и демонстрирует работу клиент-серверного взаимодействия.

---

## Цель проекта
Целью проекта является разработка простого веб-приложения, которое:
- отображает прогресс студентов;
- рассчитывает средний балл и динамику успеваемости;
- предоставляет удобную визуализацию данных;
- позволяет экспортировать результаты.

---

## Используемые технологии

### Backend
- Python
- FastAPI
- Pandas
- REST API

### Frontend
- Streamlit
- Plotly
- Requests

### Форматы данных
- CSV
- JSON

---

## Функциональные возможности

### Backend
- Получение списка студентов
- Получение списка предметов
- Получение данных о прогрессе студента
- Расчет метрик:
  - средний балл;
  - тренд успеваемости

### Frontend
- Выбор студента и предмета
- Отображение интерактивных графиков
- Просмотр рассчитанных метрик
- Экспорт данных в CSV

---

## Unit-тестирование
В проекте реализованы unit-тесты для backend части с использованием библиотеки pytest.  
Тесты проверяют корректность работы API и расчет метрик.
