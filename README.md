# Веб-приложение для визуализации прогресса студентов

# Student Progress Visualization

## Описание проекта
Данный проект представляет собой веб-приложение для визуализации и анализа учебного прогресса студентов.  
Приложение позволяет получать данные об успеваемости, рассчитывать основные метрики и отображать их в виде интерактивных графиков.

Проект реализован в виде клиент-серверного приложения и предназначен для демонстрации навыков разработки backend, frontend, тестирования и CI/CD.

---

## Цель проекта
Целью проекта является создание простого и наглядного инструмента для:
- анализа прогресса студентов;
- визуализации оценок;
- расчета среднего балла и динамики успеваемости;
- демонстрации автоматического тестирования и CI/CD.

---

## Используемые технологии

### Backend
- Python
- FastAPI
- Pandas
- REST API

### Frontend
- Streamlit
- Plotly

### Тестирование и CI/CD
- pytest
- GitHub Actions

---

## Unit-тестирование
В проекте реализованы unit-тесты для backend части с использованием библиотеки pytest.  
Тесты проверяют корректность работы API и расчет метрик.

---

## Функциональные возможности

### Backend
- Получение списка студентов
- Получение списка предметов
- Получение данных об успеваемости
- Расчет метрик:
  - средний балл;
  - тренд успеваемости

### Frontend
- Выбор студента и предмета
- Отображение интерактивных графиков
- Просмотр рассчитанных метрик
- Экспорт данных
