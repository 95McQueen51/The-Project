# Архитектура проекта

Проект реализован по клиент-серверной архитектуре.

## Backend
- Реализован с использованием FastAPI
- Предоставляет REST API
- Отвечает за обработку данных и расчет метрик

## Frontend
- Реализован с использованием Streamlit
- Получает данные через HTTP-запросы
- Отображает данные в виде интерактивных графиков

## Хранение данных
- Данные хранятся в CSV-файле
- Используется библиотека Pandas для обработки данных
